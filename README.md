# Darts Scorer App
To be run on a raspberrypi zero and allow for quick and easy calculation and scoring during a darts game between 2 players.

Instead of buying a £100 scorer, I'm going to write my own.

## Features
- simple scoring with numbered buttons
- input box to show the inputted number before pressing enter
- 301 or 501 scoring
- 2 player
- help button that gives checkout suggestions based on current score
- clear/reset scores
- error button (goes back to previous score for that player)
- legs and sets for each player (bo3 or bo5)
